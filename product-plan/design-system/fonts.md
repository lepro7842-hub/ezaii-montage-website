# Typography Configuration

## Google Fonts Import

Add to your HTML `<head>`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;500;600;700&family=Inter:wght@300;400;500;600;700&family=JetBrains+Mono:wght@400;500&family=Righteous&display=swap" rel="stylesheet">
```

## Font Usage

- **Headings:** Space Grotesk — Used for section titles, navigation, brand name
- **Body text:** Inter — Used for descriptions, paragraphs, UI text
- **Code/technical:** JetBrains Mono — Used for code snippets, conditions display
- **Brand display:** Righteous — Used exclusively for the "EZAII" brand name in the Hero section
