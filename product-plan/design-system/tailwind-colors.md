# Tailwind Color Configuration

## Color Choices

- **Primary:** `blue` — Used for buttons, links, key accents, CTA elements
- **Secondary:** `slate` — Used for tags, highlights, secondary elements
- **Neutral:** `slate` — Used for backgrounds, text, borders

## Usage Examples

Primary button: `bg-blue-600 hover:bg-blue-700 text-white`
Secondary badge: `bg-slate-100 text-slate-800 dark:bg-slate-800 dark:text-slate-200`
Neutral text: `text-slate-600 dark:text-slate-400`
Background: `bg-slate-950 dark:bg-slate-950`
Border: `border-slate-700/50`
